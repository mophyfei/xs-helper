# 上漲家數 (報價欄位)

## 語法

```xs
Value1 = GetQuote("上漲家數");
Value1 = GetQuote("UpSecurities");
Value1 = GetQuote("即時上漲家數");
Value1 = q_UpSecurities;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 市場統計 |
| 單位 | 家 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 大盤 類股指數 |

## 說明

指數成分股內目前上漲的家數。也可以使用GetField("上漲家數", "D")來表示。

大盤商品的統計會扣除權證商品。

請注意：上漲家數與漲停家數是分別統計的，上漲家數並不包含漲停的家數。
