# 總量(日) (報價欄位)

## 語法

```xs
Value1 = GetQuote("總量(日)");
Value1 = GetQuote("DailyVolume");
Value1 = q_DailyVolume;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 常用 |
| 單位 | 台股:張；大盤/指數:元；期貨/選擇權:口；其他市場股票:股 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 全部 |

## 說明

當日的總成交量。也可以使用 GetField("Volume", "D") 來表示。  
數值單位依照商品類型而異。
