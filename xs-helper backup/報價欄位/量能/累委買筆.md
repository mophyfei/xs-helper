# 累委買筆 (報價欄位)

## 語法

```xs
Value1 = GetQuote("累委買筆");
Value1 = GetQuote("CulBidTicks");
Value1 = GetQuote("累計買筆");
Value1 = q_CulBidTicks;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | 筆 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 大盤 期貨 選擇權 |

## 說明

開盤迄今的總委買筆數。也可以使用 GetField("累委買筆", "D") 來表示。

證券市場的數值為扣除權證的統計。
