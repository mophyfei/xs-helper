# 開盤賣均 (報價欄位)

## 語法

```xs
Value1 = GetQuote("開盤賣均");
Value1 = GetQuote("SoldAverageAtOpen");
Value1 = q_SoldAverageAtOpen;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 大盤 期貨 選擇權 |

## 說明

開盤時每筆委託賣出張數/口數。
計算公式：開盤委賣／開盤賣筆。

大盤的數值是整體市場扣除權證後的數值。
