# 執行比例 (報價欄位)

## 語法

```xs
Value1 = GetQuote("執行比例");
Value1 = GetQuote("ContractRatio");
Value1 = q_ContractRatio;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 期權 |
| 單位 | 值 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 台(權證) |

## 說明

回傳權證商品的執行比例。格式是數值，例如執行比例為0.5時則回傳0.5。也可以使用GetSymbolInfo("執行比例")來表示。

若認購權證執行比例為0.5，表示一張權證到期時可用履約價格購買0.5張股票。也就是每2張權證可認購1張標的股。
