# 理論價 (報價欄位)

## 語法

```xs
Value1 = GetQuote("理論價");
Value1 = GetQuote("TheoreticalPrice");
Value1 = q_TheoreticalPrice;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 期權 |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 台(權證) 選擇權 |

## 說明

選擇權／權證商品的理論價格，採用Black-Scholes模型計算。也可以使用 GetField("理論價", "D") 來表示。

可以比對現價與理論價的差異，判斷目前價格的乖離程度。
