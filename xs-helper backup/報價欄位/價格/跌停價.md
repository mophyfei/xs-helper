# 跌停價 (報價欄位)

## 語法

```xs
Value1 = GetQuote("跌停價");
Value1 = GetQuote("DailyDownlimit");
Value1 = GetQuote("今日跌停價");
Value1 = q_DailyDownlimit;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 價格 |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 台股 期貨 選擇權 大陸股票 |

## 說明

當日的跌停價。也可以使用 GetField("跌停價", "D") 來表示。
