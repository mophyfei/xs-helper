# 營益率 (報價欄位)

## 語法

```xs
Value1 = GetQuote("營益率");
Value1 = GetQuote("OpeProfitMarginRate");
Value1 = q_OpeProfitMarginRate;
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 警示 交易 函數 |
| 支援商品 | 台股 香港股票 大陸股票 美(股票) |

## 說明

最新一期財報的「營業利益率」欄位。單位為％，例如營業利益率為25%時則回傳25。

台股，陸股，港股採用的是累計財報的數值，而美股則是單季財報的數值。
