# 除息日期 (資料欄位)

## 語法

```xs
Value1 = GetField("除息日期");
Value1 = GetField("ExDividendDate");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 事件 |
| 單位 | 日期 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 說明

最近一次公佈的除息日期。

欄位格式為西元年月日，例如20150618。

請注意：這個日期可能是「未來」的日期(例如剛公佈要除息，可是尚未除息)，也可能是「過去」的日期(例如已經除完息，可是尚未公佈下次除息的日期)。

可以檢查GetFieldDate("除息日期")是否等於0來判斷是否有資料。
