# 減資新股上市日 (資料欄位)

## 語法

```xs
Value1 = GetField("減資新股上市日");
Value1 = GetField("LaunchDayOfCapitalReduction");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 事件 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 說明

最近一次減資的新股上市日。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("減資新股上市日")是否等於0來判斷是否有減資的資料。
