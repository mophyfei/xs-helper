# 理論價 (資料欄位)

## 語法

```xs
Value1 = GetField("理論價");
Value1 = GetField("TheoreticalPrice");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 期權 |
| 單位 | 點 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 分鐘 日 週 月 |
| 支援商品 | 台(權證) 選擇權 |

## 說明

選擇權／權證商品的理論價格，採用Black-Scholes模型計算。

可以比對現價與理論價的差異，判斷目前價格的乖離程度。

週頻率以上為統計期間的期底值。
