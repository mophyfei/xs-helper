# 資金流向 (資料欄位)

## 語法

```xs
Value1 = GetField("資金流向");
Value1 = GetField("CashDirect");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 週 月 還原日 還原週 還原月 季 半年 年 |
| 支援商品 | 台股 類股指數 |

## 說明

成交值占對應大盤的成交比重(％)。
單位是%，例如數值是5.6％的話，則回傳5.6。

支援台股以及類股指數，計算公式是成交值／對應大盤成交值＊100％。
