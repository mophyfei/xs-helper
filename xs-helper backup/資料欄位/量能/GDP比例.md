# GDP比例 (資料欄位)

## 語法

```xs
Value1 = GetField("GDP比例");
Value1 = GetField("ToGDPRatio");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | % |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 週 月 季 半年 年 |
| 支援商品 | 大盤 類股指數 |

## 說明

指數市值佔GDP的比例。單位是％，如果佔比是150％的話，則回傳150。

市值佔GDP比例可以用來觀察股市的熱度。計算公式：成分股總市值 / GDP ＊ 100%。

週以上頻率，是取期底值。
