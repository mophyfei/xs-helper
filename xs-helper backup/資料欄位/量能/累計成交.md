# 累計成交 (資料欄位)

## 語法

```xs
Value1 = GetField("累計成交");
Value1 = GetField("MatchUnit");
Value1 = GetField("成交張數");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 分鐘 日 |
| 支援商品 | 大盤 |

## 說明

開盤迄今整體市場(扣除權證)的總成交張數。

交易所盤中會定時公布此資訊。如果是分鐘頻率的話會取得當下最新數值，如果是日頻率的話則是當日最後的數值。

請注意此數值與大盤成交量(金額)是不一樣的統計數據。
