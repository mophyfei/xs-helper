# 委買均 (資料欄位)

## 語法

```xs
Value1 = GetField("委買均");
Value1 = GetField("AvgLongShare");
Value1 = GetField("AvgLongUnits");
Value1 = GetField("均買張數");
Value1 = GetField("均買口數");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | 大盤:張；期貨/選擇權:口 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 |
| 支援商品 | 大盤 期貨 選擇權 |

## 說明

每一筆委買的平均張數/口數。  
計算公式：累計委買／累委買筆。
