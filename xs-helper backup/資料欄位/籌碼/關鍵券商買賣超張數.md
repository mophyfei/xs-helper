# 關鍵券商買賣超張數 (資料欄位)

## 語法

```xs
Value1 = GetField("關鍵券商買賣超張數");
Value1 = GetField("KeyBrokerDifference");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 類股指數 |

## 說明

「關鍵券商」買進張數 － 賣出張數。

關鍵券商是個股實施庫藏股期間買超最大的券商分點，以此來追蹤大股東的進出動向。

類股商品（細產業指標、主題指標）計算方式為成分股加總。

請注意：因為統計時是以股為單位，所以欄位數值可能會包含小數(未滿1張時)。

資料從2013年開始提供。
