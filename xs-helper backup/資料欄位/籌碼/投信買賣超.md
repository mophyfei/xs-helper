# 投信買賣超 (資料欄位)

## 語法

```xs
Value1 = GetField("投信買賣超");
Value1 = GetField("Sdifference");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 台股,類股: 張. 大盤: 元 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 週 月 還原日 還原週 還原月 季 半年 年 |
| 支援商品 | 台股 大盤 類股指數 |

## 說明

投信買賣超。數值內容/單位依商品類型而異。

台股商品的數值為投信買張 ─ 投信賣張。
類股商品（細產業指標、主題指標）的數值則是成分股的投信買賣超的加總。
大盤商品的數值則是投信買進金額 ─ 投信賣出金額，單位是元。

週頻率以上為期間累計值。
