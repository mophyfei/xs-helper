# 內部人持股比例 (資料欄位)

## 語法

```xs
Value1 = GetField("內部人持股比例");
Value1 = GetField("Insidersharesheldratio");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

公司內部人持股占公司股本的比例。

計算公式為內部人持股張數／公司股本＊100%。

內部人統計的範圍包含董監以及公司內部的經理人。

資料每月更新一次。
