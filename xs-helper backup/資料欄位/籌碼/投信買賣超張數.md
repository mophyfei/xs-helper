# 投信買賣超張數 (資料欄位)

## 語法

```xs
Value1 = GetField("投信買賣超張數");
Value1 = GetField("SdifferenceonStock");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 週 月 還原日 還原週 還原月 季 半年 年 |
| 支援商品 | 台股 類股指數 |

## 說明

投信買賣超張數。

台股商品的數值為投信買張 ─ 投信賣張。
類股商品（細產業指標、主題指標）的數值則是成分股的投信買賣超的加總。

週頻率以上為期間累計值。
