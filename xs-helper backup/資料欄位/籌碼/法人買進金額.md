# 法人買進金額 (資料欄位)

## 語法

```xs
Value1 = GetField("法人買進金額");
Value1 = GetField("InvestorTotalBuyAmount");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 指標 警示 交易 函數 |
| 可用頻率 | 日 週 月 季 半年 年 |
| 支援商品 | 大盤 |

## 說明

三大法人(外資、投信、自營商)合計買進金額。
計算公式：外資買進金額＋投信買進金額＋自營商買進金額。

週頻率以上為期間累計值。
