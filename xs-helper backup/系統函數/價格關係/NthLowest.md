# NthLowest -  (系統函數)（價格關係）

## 語法

計算序列資料的第N個極小值。  
回傳數值=NthLowest(第幾個極小值,數列,期數)  
傳入三個參數:  
- 第一個參數是要計算極小值的序號。  
- 第二個參數是數列，通常是開高低收的價格數列。  
- 第三個參數是期數。

## 說明

以最新一筆資料為基準點，輸入要計算的期數，然後計算過去期數的第N個極小值。

範例：

```xs
value1 = NthLowest(2,low,5); //計算近5期次低的最低價
plot1(value1);               //繪製近5期次低的最低價的連線
```
