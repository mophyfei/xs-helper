# VAO -  (系統函數)（技術指標）

## 語法

計算成交量累散佈擺盪指標。  
回傳數值=VAO

## 說明

VAO指標的原文是Volume Accumulation／Distribution Oscillator指標，按照原文的意思直譯為「成交量累積散佈擺盪」指標。累積指的自然是籌碼集中累積到少數人手上的現象，而散佈自然是指籌碼散佈分散到多數人手上的現象。這個指標與VA指標（即修正OBV指標）幾乎雷同，唯一的差別是VA指標要逐日累加，VAO不累加而已。

範例：

```xs
value1 = VAO;       //計算成交量累散佈擺盪指標
plot1(value1, "VAO");
```
