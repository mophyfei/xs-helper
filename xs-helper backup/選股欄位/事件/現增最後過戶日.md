# 現增最後過戶日 (選股欄位)

## 語法

```xs
Value1 = GetField("現增最後過戶日");
Value1 = GetField("LastTransferDayOfCashIncrement");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 事件 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 說明

最近一次現金增資的最後過戶日。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("現增最後過戶日")是否等於0來判斷是否有現金增資的資料。
