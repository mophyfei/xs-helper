# 庫藏股開始日期 (選股欄位)

## 語法

```xs
Value1 = GetField("庫藏股開始日期");
Value1 = GetField("Stockstartdate");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 事件 |
| 單位 | 日期 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台股 |

## 說明

最新一次庫藏股預計買回期間的開始日期。

欄位格式為西元年月日，例如20221101。

可以檢查GetFieldDate("庫藏股開始日期")是否等於0來判斷是否有資料。
