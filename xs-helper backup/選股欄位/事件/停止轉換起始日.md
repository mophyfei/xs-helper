# 停止轉換起始日 (選股欄位)

## 語法

```xs
Value1 = GetField("停止轉換起始日");
Value1 = GetField("StopConvertingSD");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 事件 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 單一 |
| 支援商品 | 台(可轉債) |

## 說明

最近一次停止轉換起始日的日期。

欄位格式為西元年月日，例如20250818

請注意：這個日期可能是「未來」的日期（例如剛公告債券停止轉換起始日，可是尚未進入轉換停止期間）也可能是「過去」的日期（例如已經進入停止轉換，或是已經結束停止轉換）

可以檢查GetFieldDate("停止轉換起始日")是否等於0來判斷是否有資料。

停止受理轉換(停止轉換)之事由通常為：辦理(1)配股配息，(2)股東常會，(3)股東臨時會，(4)現金增資，(5)減資 等作業事宜。
