# 借款依存度 (選股欄位)

## 語法

```xs
Value1 = GetField("借款依存度");
Value1 = GetField("Debt／Eqity ％");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 |

## 說明

財務比率表上的「借款依存度」欄位。  
提供單季與年度資料。  
單位為%，例如：借款依存度為150%，則回傳150。  
不提供此欄位的公司，輸出值會以零表示。
