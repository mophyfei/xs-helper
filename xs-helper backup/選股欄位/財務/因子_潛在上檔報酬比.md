# 因子_潛在上檔報酬比 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_潛在上檔報酬比");
Value1 = GetField("Factor_UPR");
Value1 = GetField("因子_潛在上檔報酬比率因子");
Value1 = GetField("Factor_Upside Potential Ratio Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

將過去1年的日報酬率隨機抽樣1000筆，以正報酬率的加總與過去1年日交易天數的比值作為分子，以負報酬率的平方加總與過去1年日交易天數的比值平方根作為分母，得到的比率即為潛在上檔報酬比率因子。
