# 因子分數_半年動能因子 (選股欄位)

## 語法

```xs
Value1 = GetField("因子分數_半年動能因子");

Value1 = GetField("因子分數_動能因子(以前7~12月報酬為基準)");
Value1 = GetField("FactorScore_Momentum Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
