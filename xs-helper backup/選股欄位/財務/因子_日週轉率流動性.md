# 因子_日週轉率流動性 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_日週轉率流動性");
Value1 = GetField("Factor_TURNOVER_AVG");
Value1 = GetField("因子_流動性因子(以平均日週轉率為基準)");
Value1 = GetField("Factor_Trading volume Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

過去一個月的日週轉率平均值。
日週轉率=日成交量/發行量。
