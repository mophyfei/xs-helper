# 其他資產 (選股欄位)

## 語法

```xs
Value1 = GetField("其他資產");
Value1 = GetField("TotalOtherAssets");
Value1 = GetField("其他非流動資產");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 |

## 說明

一般、證券股:「資產負債表」中的「其他非流動資產」項目。  
金融、壽險、產險:「資產負債表」中的「其他資產」項目。
