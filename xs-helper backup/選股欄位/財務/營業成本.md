# 營業成本 (選股欄位)

## 語法

```xs
Value1 = GetField("營業成本");
Value1 = GetField("CostOfGoodsSold");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 美(股票) |

## 說明

凡銷售貨物、提供勞務及金融保險業務等各項成本皆屬之。
台股:「損益表」中的「營業成本」。
美股:「損益表」中「銷貨成本」項目。
台股中金融、證券股不適用此欄位。
