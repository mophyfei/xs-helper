# 因子_半年動能因子 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_半年動能因子");
Value1 = GetField("Factor_MOM_1Y");
Value1 = GetField("因子_動能因子(以前7~12月報酬為基準)");
Value1 = GetField("Factor_Momentum Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

股票在過去1年至過去半年這段期間內(12個月前至7個月前)的漲跌幅度。
