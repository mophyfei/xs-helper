# 短期投資 (選股欄位)

## 語法

```xs
Value1 = GetField("短期投資");
Value1 = GetField("MarketableSecurity");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 百萬 |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 |

## 說明

「資產負債表」中「流動資產」項內的「"透過損益按公允價值衡量之金融資產－流動" + "透過其他綜合損益按公允價值衡量之金融資產－流動" + "按攤銷後成本衡量之金融資產－流動"」三個項目加總。
