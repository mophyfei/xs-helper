# 因子_淨營運資產 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_淨營運資產");
Value1 = GetField("Factor_NOA");
Value1 = GetField("因子_淨營運資產因子");
Value1 = GetField("Factor_Net Operating Assets Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

淨營運資產/去年同期總資產。
淨營運資產=營運資產-營運負債。
營運資產=資產總額-現金及短期投資。
營運負債=資產總額-短期負債-長期負債-少數股權-特別股股本-普通股股本。
