# 因子_前三年獲利 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_前三年獲利");
Value1 = GetField("Factor_PROFITABILITY_LAG3");
Value1 = GetField("因子_獲利因子(以前三年獲利為基準)");
Value1 = GetField("Factor_three year lagged  profitability Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

三年前的營業利益/三年前的帳面權益。
帳面權益=總資產-總負債+遞延所得稅-特別股股本。
