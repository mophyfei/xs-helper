# 因子_一年動能 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_一年動能");
Value1 = GetField("Factor_MOM");
Value1 = GetField("因子_動能因子(以前11個月報酬為基準)");
Value1 = GetField("Factor_Momentum Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

股票在過去1年內(不包含當月，共計11個月)的漲跌幅度。
