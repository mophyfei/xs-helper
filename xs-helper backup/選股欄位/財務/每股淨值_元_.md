# 每股淨值(元) (選股欄位)

## 語法

```xs
Value1 = GetField("每股淨值(元)");
Value1 = GetField("BookValuePerShare");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 台股：元。美股：元(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 美(股票) |

## 說明

台股，是財務比率表的「每股淨值(F)(TSE公告數)」項目。  
美股，是財務比率表上的「每股淨值」項目。  
提供單季與年度資料。
