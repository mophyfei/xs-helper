# 盈餘營收成長率比 (選股欄位)

## 語法

```xs
Value1 = GetField("盈餘營收成長率比");
Value1 = GetField("EarningGrowth／SalesGrowth");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 |

## 說明

計算公式: 本期稅後淨利年成長率 / 營業收入淨額年成長率 \*100%  
提供單季與年度資料。  
單位為%，例如：欄位數值為25%，則回傳25。
