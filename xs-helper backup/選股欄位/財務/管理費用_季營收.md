# 管理費用／季營收 (選股欄位)

## 語法

```xs
Value1 = GetField("管理費用／季營收");
Value1 = GetField("OperatingExpense－ADM／Sales");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 |
| 支援商品 | 台股 |

## 說明

計算公式: 營業費用的管理費用/營業收入淨額\*100%。  
提供單季與年度資料。  
單位為%，例如：欄位數值5%，則回傳5。不提供此欄位的公司，輸出值會以零表示。
