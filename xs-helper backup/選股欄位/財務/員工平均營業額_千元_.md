# 員工平均營業額(千元) (選股欄位)

## 語法

```xs
Value1 = GetField("員工平均營業額(千元)");
Value1 = GetField("SalesPerEmployee");
Value1 = GetField("每人營收");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 千元 |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 |

## 說明

財務比率表內的「每人營收」欄位。  
計算公式 : 營業收入／平均員工人數
