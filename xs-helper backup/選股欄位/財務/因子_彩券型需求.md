# 因子_彩券型需求 (選股欄位)

## 語法

```xs
Value1 = GetField("因子_彩券型需求");
Value1 = GetField("Factor_MAX");
Value1 = GetField("因子_彩券型需求因子");
Value1 = GetField("Factor_Lottery Demand Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

過去一個月中最大五筆日報酬率的簡單算數平均。
