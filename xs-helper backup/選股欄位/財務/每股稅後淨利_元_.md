# 每股稅後淨利(元) (選股欄位)

## 語法

```xs
Value1 = GetField("每股稅後淨利(元)");
Value1 = GetField("EPS");
Value1 = GetField("EarningPerShare");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 台股：元。美股：元(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 美(股票) |

## 說明

台股，是損益表上的「每股盈餘」項目。  
美股，是損益表上的「基本每股盈餘(由淨利計算)」項目。  
提供單季與年度資料。
