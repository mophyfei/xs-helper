# 因子分數_股票發行量 (選股欄位)

## 語法

```xs
Value1 = GetField("因子分數_股票發行量");
Value1 = GetField("FactorScore_ISSUE");
Value1 = GetField("因子分數_股票發行量因子");
Value1 = GetField("FactorScore_Share issuance Factor");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 月 |
| 支援商品 | 台股 |

## 說明

股票發行量因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。
