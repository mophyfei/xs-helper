# 無形資產 (選股欄位)

## 語法

```xs
Value1 = GetField("無形資產");
Value1 = GetField("IntangibleAssets");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 財務 |
| 單位 | 台股：百萬。美股：百萬(美元) |
| 格式 | 數值 |
| 支援腳本 | 指標 選股 警示 交易 函數 |
| 可用頻率 | 季 年 |
| 支援商品 | 台股 美(股票) |

## 說明

台股:「資產負債表」中「非流動資產」項內「其他無形資產淨額」項目。  
美股:「資產負債表」中「非流動資產」項內「商譽與無形資產淨額」項目。  
不提供此欄位的公司，輸出值會以零表示。
