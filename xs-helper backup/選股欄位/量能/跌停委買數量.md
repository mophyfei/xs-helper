# 跌停委買數量 (選股欄位)

## 語法

```xs
Value1 = GetField("跌停委買數量",param:=整體市場);
Value1 = GetField("DownLimitBidUnits",param:=整體市場);
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 說明

TSE支援 整體市場、股票、上市創新板股票、基金、認購權證、認售權證。

OTC支援 整體市場、股票、基金、認購權證、認售權證。

可以用 GetField("跌停委買數量", "D", param:= "股票") 的方式來取得個別資訊。

若沒有提供 param 參數的話，預設回傳 整體市場 的數值。

若商品沒有對應的 param (ex. GetSymbolField("OTC.TW", "漲停委買數量", "D", param:= "上市創新板股票")) 則回不支援即可。

備註:「整體市場」為股票、上市創新板股票、基金、認購權證、認售權證及其他有價證券之合計。
