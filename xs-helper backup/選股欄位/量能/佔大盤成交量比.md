# 佔大盤成交量比 (選股欄位)

## 語法

```xs
Value1 = GetField("佔大盤成交量比");
Value1 = GetField("WeightOfTSE");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 量能 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 日 |
| 支援商品 | 台股 |

## 說明

計算公式：個股成交值 / 對應大盤指數成交值 \* 100%。

單位為%，例如比值為1.5%時，回傳值為1.5。

上市股票的對應大盤指數為加權指數，上櫃股票的對應大盤指數為櫃買指數。
