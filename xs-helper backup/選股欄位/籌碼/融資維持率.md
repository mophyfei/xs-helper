# 融資維持率 (選股欄位)

## 語法

```xs
Value1 = GetField("融資維持率");
Value1 = GetField("Pomremainratio");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | ％ |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 日 週 月 季 半年 年 |
| 支援商品 | 台股 |

## 說明

系統估算的大盤或是個股的融資維持率，也就是股票市值與融資金額的比例。

計算公式:

個股融資維持率=收盤價／(融資成本線 \* 0.6) \* 100%

大盤融資維持率=所有融資股票市值／大盤融資餘額 \* 100%

依查詢頻率取期底值。

註：

個股融資成本=(昨日融資買進成本 \* (融資餘額-融資買進) + 收盤價 \* 融資買進) / 融資餘額
