# 實戶買賣超張數 (選股欄位)

## 語法

```xs
Value1 = GetField("實戶買賣超張數");
Value1 = GetField("operatordifference");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 日 週 月 |
| 支援商品 | 台股 |

## 說明

實戶買張 ─ 實戶賣張。

實戶買張是單一價格成交值介於50萬到100萬之間的合計買進張數，而實戶賣張則是同樣級距的合計賣出張數。

週頻率以上為期間累計值。
