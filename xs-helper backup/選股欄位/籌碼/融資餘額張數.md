# 融資餘額張數 (選股欄位)

## 語法

```xs
Value1 = GetField("融資餘額張數");
Value1 = GetField("Pomremain");
```

| 屬性 | 值 |
|---|---|
| 欄位分類 | 籌碼 |
| 單位 | 張 |
| 格式 | 數值 |
| 支援腳本 | 選股 函數 |
| 可用頻率 | 日 週 月 還原日 還原週 還原月 |
| 支援商品 | 台股 類股指數 |

## 說明

融資餘額張數，也就是到目前以融資買進的淨張數。

這個數值代表著還有多少融資張數尚未償還。

類股商品（細產業指標、主題指標）的數值為成分股融資餘額張數的加總。

週頻率以上為統計期間的期底值。
